# Video-game templating

## Usage

```bash
# Install deps
yarn install

# Start loval server
yarn start
```

## Enpoints

### In web browser

- All games: [http://localhost:8080/games](http://localhost:8080/games)
- All platforms: [http://localhost:8080/platforms](http://localhost:8080/platforms)

### Rest client

- `GET /platforms`
- `GET /platforms/:slug`
- `GET /games/`
- `GET /games/:slug`
